# HOW TO INSTALL

- 1. Клонировать репозиторий **git clone https://github.com/Goffroid/PHOTO.git**    
- 2. Установить зависимости **pip install -r requirements.txt**   
- 3. Перейти в директорию **cd music_generation_project**
- 3. (Если требуется, запустить обучение) **python src/train.py**   
- 4. Сгенерировать мелодию в формате .npy **python src/generate.py**   
- 5. Преобразовать .npy в .mdi **python src/convert_to_midi.py**   

### После этого внутри папки проекта появится готовый файл с мелодией и график, который наглядно всё покажет, что за мелодия была сделана
